# 🐧 Gentoo - Hyprland

My Gentoo + Hyprland setup. Warm neutral palette, Lua config, still tweaking it here and there.

### ⚙️ Stack
- **OS:** Gentoo
- **WM:** Hyprland
- **Launcher:** Rofi
- **Wallpaper:** Hyprpaper
- **Screenlock:** Hyprlock + Hypridle
- **Terminal:** Kitty
- **Notification daemon:** Mako
- **Screenshot tool:** Hyprshot
- **Status bar:** Waybar
- **File manager:** Dolphin
- **Icons:** Papirus
- **Cursor:** rose-pine-hyprcursor
